PROJECT 1: HOSPITAL MANAGEMENT SYSTEM


Full-Stack Web Development

Developed a robust Full-Stack application using Java, Spring Boot, Spring Data JPA, and MySQL.

The Hospital Management System is designed to automate healthcare workflows, featuring an integrated MVC Architecture for seamless user experience.

Key functionalities include Doctor Onboarding, Automated Appointment Scheduling, and Digital Receipt Generation.

Utilized Hibernate for database persistence and Thymeleaf for creating a dynamic, responsive front-end interface.

PROJECT 2: AUDITORIUM MANAGEMENT SYSTEM


Backend Development with Python & Django

Developed during my MCA, this web-based application is built using Python (Django), HTML, CSS, JavaScript, and PostgreSQL.

The system efficiently manages auditorium bookings, event scheduling, and tiered user access.

It enables administrators and users to handle real-time reservations, check hall availability, and maintain digitized event records in an organized manner.
