PROJECT


Auditorium Management system

I developed this project while pursuing my MCA, using Python, Django, and MySQL.

Auditorium Management System is a web-based application developed using Python (Django), HTML, CSS, JavaScript, and PostgreSQL.

It helps manage auditorium bookings, event scheduling, and user access efficiently.

The system allows administrators and users to handle reservations, view availability,

and maintain event records in an organized and digitized manner.
